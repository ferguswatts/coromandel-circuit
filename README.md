# Coromandel E-Bike Circuit

Interactive map for a 9-day, 493km e-bike bikepacking circuit of the Coromandel Peninsula, New Zealand.

**Live site:** https://YOUR_USERNAME.github.io/coromandel-circuit/

## The Route
- **Days:** 9 riding days, 8 nights
- **Distance:** 493km
- **Surface:** Mix of sealed road, gravel, rail trail, and MTB track
- **Start/Finish:** Thames

## Stops
1. Thames → Otautu Bay Farm Camp (85km)
2. Otautu Bay → Port Jackson DOC (21km)
3. Port Jackson → Tangiaro Retreat (23km)
4. Tangiaro → Kennedy Bay (33km)
5. Kennedy Bay → Whitianga (54km)
6. Whitianga → Whitianga / 309 Road loop (68km)
7. Whitianga → Tairua (40km)
8. Tairua → Whangamata (47km)
9. Whangamata → Paeroa (81km)
10. Paeroa → Thames (41km)

## Setup
Just a single `index.html` file — no build step needed. Uses Leaflet.js via CDN with CartoDB dark tiles.

To host: enable GitHub Pages on the `main` branch in repo Settings → Pages.
